## Verwendete Prompts

1. Ich möchte gerne ein Hello World Programm in Java programmieren. Bitte lege das Programm in einem Unterverzeichnis java an.
2. Ich möchte das jetzt auch in Python haben. Lege das in einem Unterverzeichnis python an.
3. Bitte lege im python-Verzeichnis ein virtuelles Environment an.
4. Ich möchte das Hello World Java-Programm zu einem Service mit REST-Interface erweitern. Bitte benutze Spring Boot. Wenn ich mit GET die URL api/hello aufrufe, soll mit "Hello World!" geantwortet werden. Wenn ich api/hello/name aufrufe, soll mit "Hello name!" geantwortet werden.
5. Erstelle ein README im Java-Verzeichnis, in dem meine Anforderungen dokumentiert sind und die Anweisungen zur Benutzung.
6. Bitte den Namen als Request-Parameter übergeben.
7. Bitte platziere den REST-Quelltext in einer neuen Klasse HelloController.
8. Bitte benenne Attribute in Klassen gemäß der Python-Konvention mit Unterstrich.
9. Das Programm beendet sich sofort nach dem Start.
10. Kann die Java-Version auch die Requests anzeigen wie die Python-Version? (Log-Ausgaben)
11. Geht das nicht einfacher als mit dem Filter? Kann man nicht direkt im Servlet Container das Logging aktivieren?
12. Bitte aktualisiere das README im Java-Verzeichnis.
13. Bitte Anforderung Java Version 25.
14. Bei der Build-Anweisung das Kommando zum Starten weglassen, das steht ja bereits oben.
15. Bitte lege auch im Python-Verzeichnis ein README an.
16. Was bedeutet der --reload Parameter?
17. Bitte aktualisiere die globale README und verweise dort auf die beiden Unterverzeichnisse.
18. Bitte dokumentiere die verwendeten Prompts in dieser Session in einem neuen Abschnitt der README.
